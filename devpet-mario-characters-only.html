Codepet Character Sprites

Seven Super Mario-style pixel sprites, freshly redrawn. Each character is a standalone SVG rendered at native resolution with pixel-perfect crispness. Swap the source files in sprites-svg/ to reskin them.

Asset format: each sprite is an SVG built from 16×16 pixel <path> blocks. Drop a replacement SVG in sprites-svg/ under the same filename to swap a character. The browser renders with image-rendering:pixelated so upscaling stays crisp.
