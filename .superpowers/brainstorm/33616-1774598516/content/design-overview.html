6. Key Changes Summary

  • Font: Inter → Varela Round
  • Background: warm-bg #FAFFFE → clean white #FFFFFF
  • Primary: mint #34D399 kept, darker shadow #059669 added
  • Borders: soft green #D1FAE5 → neutral grey #E5E5E5
  • Buttons: flat → 3D with 5px bottom shadow + push-down
  • Cards: subtle shadow → raised 3D with border + bottom shadow
  • Text: green-tinted greys → pure greys (#4B4B4B, #777)
  • Radius: mixed → consistent 16px on cards/buttons
  • CTAs: sentence case → UPPERCASE + letter-spacing
  • Animations: smooth ease → bouncier spring physics
