Core Components

3 New/Modified Components

1. PetSlot — Invisible anchor point

  • Đặt trong mỗi section tại vị trí mong muốn
  • Register vị trí của mình vào Context khi mount
  • Dùng useInView (Framer Motion) để báo khi section visible
  • Render: chỉ một <div ref /> ẩn — zero visual footprint

2. PetGuide — The actual pet (replaces FloatingCompanion)

  • Chỉ 1 instance render trên toàn page
  • Đọc active section từ Context → lấy tọa độ PetSlot tương ứng
  • Dùng motion.div với animate để bounce đến vị trí mới
  • Render: pet sprite + speech bubble + click interaction
  • Khi chuyển section: position: fixed tạm → bounce → về absolute

3. CompanionProvider (enhanced) — State management

  • Giữ: characterName, activeSection, slotPositions
  • Thêm: registerSlot(), setActiveSection()
  • Thêm: isTransitioning flag cho animation state

Data Flow

Scroll → Pet Movement Flow

1. User scrolls → PetSlot's useInView fires

2. PetSlot calls setActiveSection("features")

3. PetGuide reads new activeSection from Context

4. PetGuide gets target position from slotPositions["features"]

5. PetGuide animates: current pos → arc up → bounce down to target

6. Speech bubble fades in after pet lands (300ms delay)

7. Pet switches to section-specific idle pose
