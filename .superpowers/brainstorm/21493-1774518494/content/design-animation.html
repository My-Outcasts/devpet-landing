Bounce Animation

Pet chuyển từ Section A → Section B
Slot A (cũ)
🐱
Slot B (mới)

Keyframes:

1. Launch (0-200ms) — Pet scale down 0.8 → nhún lên, rời slot cũ

2. Arc (200-500ms) — Bay theo đường cong lên trên (translateY giảm)

3. Land (500-700ms) — Rơi xuống slot mới với bounce spring

4. Settle (700-900ms) — Squash & stretch nhẹ khi chạm đất

5. Speech (900-1200ms) — Bubble fade in + scale from 0

Framer Motion Config

Animation Settings
// Bounce transition
transition: {
  type: "spring",
  stiffness: 300,
  damping: 20,
  mass: 0.8
}

// Squash & stretch on land
onAnimationComplete → trigger:
  scaleY: [1, 0.8, 1.1, 1]  // squash → stretch → settle
  scaleX: [1, 1.2, 0.9, 1]  // inverse
  duration: 200ms

Click Interaction

Tap/Click vào Pet

Click → Fun reaction:

1. Pet nhảy lên 1 lần (translateY -20px → spring back)

2. Xoay nhẹ random (-15° đến +15°)

3. Hiện random quote từ pool (khác speech bubble chính)

4. Sparkle particles quanh pet (3-5 dots, fade out)


Random quotes pool (ví dụ):

"Hehe!" · "Code thêm đi!" · "Bug-free zone 🛡" · "Ship it! 🚀" · "Ngon lắm!"


Cooldown: 1 giây giữa mỗi click để tránh spam animation.
